CPU Education

The CPU is considered to be the brains of a computer. Without it, the computer is unable to process and perform any tasks. When learning about computer architecture, the CPU plays an important part because it get instructions from memory, decodes it into binary, executes it, and stores the output to memory. The lesson plan is aimed towards educational providers who want to implement a beginner course on CPU development and for self-learners. It is assumed that they have basic coding skills and have Vivado, a software suite to access and experiment with our CPU design.

A breakdown of each CPU component is shown below. Each module explains what its functionalities are and and how it interacts with other components. There are also short examples and quiz questions before moving onto the next module. These modules should be used alongside this Vivado project, a simple CPU designed for others with very little knowledge on CPU design and computer architecture. Learners should be able to understand each component and implement the given examples. In addition, the documentation and examples should allow them to explore and modify the simple CPU code on their own.
